# data/ README

This directory stores **intermediate R objects** produced by the
GRAVE-D data pipeline in `R/00–05_*.R`. Files here are not meant to be
edited by hand; they are regenerated by re-running the scripts.

## Core GRAVE-D intermediates

| File name                 | Unit of analysis  | Description                                                | Produced by script             | Consumed by script(s)          |
        |---------------------------|-------------------|------------------------------------------------------------|--------------------------------|--------------------------------|
        | spine.rds                 | dyad-year         | FBIC-based dyadic spine with COW codes and basic structure | 01_build_fbic_spine.R          | 02_build_conflict.R, 03_build_grave_d_ideology.R, 04_build_controls.R, 05_build_master.R |
        | spine_conflict.rds        | dyad-year         | Spine plus MID-based conflict variables                    | 02_build_conflict.R            | 05_build_master.R              |
        | spine_ideology.rds        | dyad-year         | Spine plus leader/ideology variables (Archigos, Colgan, leader_ideology) | 03_build_grave_d_ideology.R   | 05_build_master.R              |
        | spine_controls.rds        | dyad-year         | Spine plus controls (alliances, econ, CINC, etc.)         | 04_build_controls.R            | 05_build_master.R              |
        | GRAVE_D_Master.rds        | dyad-year         | Full merged GRAVE-D dyadic dataset (R object)              | 05_build_master.R              | Internal use, export to ready_data/ |
        

To rebuild all intermediates, run the 00–05 scripts in order:

R/00_packages.R

R/01_build_fbic_spine.R

R/02_build_conflict.R

R/03_build_grave_d_ideology.R

R/04_build_controls.R

R/05_build_master.R

Do not commit hand-edited versions of these files; treat them as
generated artifacts of the pipeline.
